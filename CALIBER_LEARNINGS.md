# Caliber Learnings

Accumulated patterns and anti-patterns from development sessions.
Auto-managed by [caliber](https://github.com/caliber-ai-org/ai-setup) ‚Äî do not edit manually.

- **[gotcha]** `caliber sync` is NOT a valid subcommand ‚Äî use `caliber refresh` instead. CLAUDE.md and `.claude/rules/agent-config.md` both document `caliber sync` but the binary does not have this command (confirmed via `caliber --help`). Running `caliber sync` fails silently with empty output.
- **[pattern]** `caliber init` takes ~10 minutes ‚Äî always run as a background task: `caliber init --auto-approve --agent claude,codex`. Wait for the background task notification rather than polling the output file.
- **[gotcha]** `caliber refresh --quiet` produces no stdout on success ‚Äî empty result is normal. Verify via exit code or background task notification, not visible output.
- **[env]** MCP packages `@modelcontextprotocol/server-filesystem` and `@modelcontextprotocol/server-github` are installed globally at `/opt/homebrew/lib`. The `npx -y` in `.mcp.json` uses cached global versions without a network fetch.
- **[convention]** `statusLine` config belongs in `~/.claude/settings.json` (global user-level), NOT project `.claude/settings.json`. Project settings manages hooks; user settings manages UI config.
- **[env]** `.caliber/` directory is gitignored ‚Äî caliber state (including `lastRefreshSha` for freshness hooks) is machine-local, never shared via git.
- **[gotcha]** `caliber init` output file contains raw ANSI escape sequences (spinner animations `\u001b[0J`, `\u001b[8A`, etc.) ‚Äî grepping the output file for completion markers is unreliable. Use `pgrep -f caliber` to check if still running, or wait for background task notification.
- **[gotcha]** Files generated by `caliber init` ‚Äî including `.claude/rules/agent-config.md` and `AGENTS.md` ‚Äî contain stale `caliber sync` references. After every `caliber init`, check these files and replace `caliber sync` with `caliber refresh`.
- **[gotcha]** `caliber score --json` produces multi-line JSON ‚Äî piping through `head -1` returns only `{`. Read the full output or omit `head -1`.
- **[gotcha]** `caliber refresh` may emit "Last refresh failed: Claude Code CLI is installed but not logged in" warning at the top of output ‚Äî this is non-fatal; the refresh still executes and updates all configs. Don't treat as a blocking error.
- **[env]** MCP github server requires `GITHUB_TOKEN` exported in the shell ‚Äî `.mcp.json` uses `${GITHUB_TOKEN}` env substitution. If unset, all `mcp__github__*` tools will fail. Export before starting Claude Code: `export GITHUB_TOKEN=<pat>`.
